# CodeAlpha_AI-chatbot
# 🤖 Basic Chatbot using Python

## 📌 Project Overview

This project was developed as **Task 2** for the **CodeAlpha Python Programming Internship**.

It is a simple **rule-based chatbot** built using Python. The chatbot interacts with users by responding to predefined questions and greetings using conditional statements. It demonstrates the fundamentals of conversational programming without using AI or machine learning.

---

## 🚀 Features

- 👋 Greets users and starts a conversation
- 💬 Responds to common questions and greetings
- 😂 Tells a simple programming joke
- 😊 Responds to user emotions like "I am happy" and "I am sad"
- 🙏 Handles gratitude with friendly responses
- 🚪 Ends the conversation when the user types **"bye"**
- ⚠️ Displays a default message for unknown inputs

---

## 🛠️ Technologies Used

- Python 3
- Conditional Statements (`if`, `elif`, `else`)
- Loops (`while`)
- User Input (`input()`)
- Functions
